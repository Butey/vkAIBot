# vkAIBot

Бот для работы с LLM моделями через ВКонтакте.

## Описание

Этот бот позволяет пользователям взаимодействовать с языковыми моделями (LLM) через сообщения ВКонтакте. Бот поддерживает контекст диалога и использует OpenAI API для генерации ответов.

## Требования

- Python 3.8+
- Токен сообщества ВКонтакте
- API ключ OpenAI

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd vkAIBot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните файл `.env` своими данными:
- `VK_TOKEN` - токен вашего сообщества ВКонтакте
- `OPENAI_API_KEY` - ваш API ключ OpenAI
- `OPENAI_MODEL` - модель OpenAI (по умолчанию: gpt-3.5-turbo)

## Запуск

```bash
python bot.py
```

## Структура проекта

- `bot.py` - основной файл бота
- `requirements.txt` - список зависимостей
- `.env.example` - шаблон файла с переменными окружения
- `README.md` - документация

## Возможности

- Поддержка контекста диалога (последние 10 сообщений)
- Интеграция с OpenAI API
- Long polling для получения сообщений
- Логирование событий
